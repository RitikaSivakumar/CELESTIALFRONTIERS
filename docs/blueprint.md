# **App Name**: WellGuard AIoT

## Core Features:

- Wearable Sensor Data Ingestion: Continuously collect physiological and behavioral signals from the AIoT wearable device.
- Real-time Health Assessment: Analyze sensor data and user inputs for ongoing assessment of mental and physical state.
- Age-Based Personalization: Display personalized insights and suggested interventions based on age group.
- Doctor Assignment & Reporting: Assign users to doctors and securely provide doctors access to patient health reports, with data privacy controls.
- Smart Interventions & Notifications: Trigger notifications with positive messages to reduce stress and improve well-being.
- AI Chatbot for Emotional Support: Engage users in conversations via text, voice, and video, and apply emotion analysis to respond supportively.
- Psychological Engagement Features: Show positive and uplifting messages and encourage interaction with wellness-focused content and games.

## Style Guidelines:

- Primary color: Soft sky blue (#87CEEB) to evoke calmness and tranquility.
- Background color: Light gray (#F5F5F5) for a clean and neutral backdrop.
- Accent color: Pale lavender (#E6E6FA) to enhance feelings of relaxation and well-being.
- Body and headline font: 'PT Sans', a modern yet friendly sans-serif, for readability.
- Use minimalist icons to represent different features, with calming pastel shades.
- Clean and simple layout, promoting ease of navigation and reducing cognitive load.
- Subtle animations to indicate changes, maintaining a non-intrusive and relaxing user experience.
